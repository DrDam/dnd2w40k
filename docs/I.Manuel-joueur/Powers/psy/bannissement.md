#### Bannissement {#bannissement}

*Pouvoir de niveau 4*

**Temps d'incantation** : 1 action

**Portée** : 18 mètres

**Durée** : concentration, jusqu'à 1 minute

Vous tentez d'envoyer une créature que vous pouvez voir à portée dans l'immaterium. La cible doit réussir un jet de sauvegarde de Charisme ou être bannie.

Si la cible se trouve déjà dans l'immaterium, vous la bannissez vers un autre demi-plan de l'immaterium. Là-bas, elle est neutralisée. Elle y demeure jusqu'à la fin du pouvoir, moment auquel elle réapparaît dans l'espace qu'elle a quitté, ou dans l'espace inoccupé le plus proche si cet espace est occupé.

Si la cible se trouve dans l'espace réel, elle est bannie et retourne dans l'immaterium. Si le pouvoir prend fin avant qu'une minute ne se soit écoulée, la cible réapparaît dans l'espace qu'elle a quitté, ou dans l'espace inoccupé le plus proche si cet espace est occupé. Sinon, elle ne revient pas.

**Aux niveaux supérieurs.** Lorsque vous lancez ce pouvoir au niveau 5 ou supérieur, vous pouvez cibler une créature supplémentaire pour chaque niveau au-dessus du niveau 4.
