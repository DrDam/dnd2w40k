#### Image miroir {#image-miroir}

*Pouvoir de niveau 2*

**Temps d'incantation** : 1 action

**Portée** : soi-même

**Durée** : 1 minute

**Trois doubles** illusoires de vous-même apparaissent dans votre espace. Jusqu'à la fin du pouvoir, les doubles se déplacent avec vous et imitent vos actions, changeant de position de sorte qu'il est impossible de déterminer quelle image est réelle. Vous pouvez utiliser votre action pour dissiper les doubles illusoires.

Chaque fois qu'une créature vous cible avec une attaque pendant la durée du pouvoir, lancez un d20 pour déterminer si l'attaque cible à la place l'un de vos doubles.

Si vous avez trois doubles, vous devez obtenir 6 ou plus pour rediriger l'attaque vers un double. Avec deux doubles, vous devez obtenir 8 ou plus. Avec un seul double, vous devez obtenir 11 ou plus.

La CA d'un double est égale à 10 + votre modificateur de Dextérité. Si une attaque touche un double, celui-ci est détruit. Un double ne peut être détruit que par une attaque qui le touche. Il ignore tous les autres dégâts et effets. Le pouvoir prend fin lorsque les trois doubles sont détruits.

Une créature n'est pas affectée par ce pouvoir si elle ne peut pas voir, si elle se fie à des sens autres que la vue, comme la "vision de l'aveugle", ou si elle peut percevoir les illusions comme telles, comme avec la "vision véritable".
