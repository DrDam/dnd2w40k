#### Vision dans le noir {#vision-dans-le-noir-tech-power}

*Pouvoir de niveau 2*

**Temps d'incantation** : 1 action

**Portée** : contact

**Durée** : 8 heures

Vous touchez une créature consentante pour lui accorder la capacité de voir dans le noir. Pour la durée du pouvoir, cette créature possède la vision dans le noir jusqu'à une portée de 18 mètres.
