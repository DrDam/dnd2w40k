#### Médication parfaite {#medication-parfaite}

*Pouvoir de niveau 9*

**Temps d'incantation** : 1 action

**Portée** : contact

**Durée** : instantanée

Vous administrez le remède parfait à une créature que vous touchez. La cible récupère la totalité de ses points de vie. Si la créature est charmée, effrayée, paralysée ou étourdie, la condition prend fin. Si la créature est à terre, elle peut utiliser sa réaction pour se relever. Ce pouvoir n'a aucun effet sur les morts-vivants ou les constructions.
